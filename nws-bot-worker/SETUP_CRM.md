# Настройка CRM (база клиентов через группу с темами)

## 1. Создайте группу

- Создайте **закрытую** Telegram-группу
- Добавьте всех менеджеров
- Добавьте бота @YourBotName

## 2. Включите темы (форум)

- Настройки группы → **Темы** (Topics) → **Включить**
- Группа станет форумом с разделами

## 3. Права бота

- Сделайте бота **администратором**
- Включите право **«Управление темами»** (Manage topics)
- Включите право **«Закрепление сообщений»** (Pin messages)

## 4. Узнайте ID группы

- Добавьте @userinfobot в группу — он покажет ID
- Или перешлите любое сообщение из группы боту @getidsbot
- ID группы выглядит как `-1001234567890`

## 5. Укажите GROUP_ID

В Cloudflare Dashboard → Workers & Pages → nwsnumbot → **Settings** → **Variables**:

- Добавьте переменную `GROUP_ID` = ваш ID группы (например `-1001234567890`)

Или в `wrangler.toml` в секции `[vars]`:
```toml
GROUP_ID = "-1001234567890"
```

## 6. Деплой

```powershell
cd nws-bot-worker
npx wrangler deploy
```

## Как это работает

- **Новый клиент** пишет боту → создаётся тема с именем «Имя | @username | ID: 12345»
- **Сообщения клиента** → пересылаются в его тему
- **Ответ менеджера** в теме → пересылается клиенту от лица бота
- **Заказ через Web App** → детали в тему + закрепление (🔴 Не оплачен)
- **Без GROUP_ID** → всё идёт в личку менеджеру (старое поведение)
